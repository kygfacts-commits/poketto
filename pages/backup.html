Backup & Restore

Kelola data keuangan kamu

Restore menggabungkan data: baris dengan ID yang sama di file backup akan menimpa data yang ada, sedangkan data lain yang tidak ada di file backup tetap dipertahankan (tidak dihapus). Pastikan file backup valid.

Export Data

Unduh seluruh datamu (akun, transaksi, kategori, budget, goals, tagihan, portfolio) sebagai satu file JSON.

Import Data

Pilih file backup JSON lalu klik Restore.
